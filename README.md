# ОПИСАНИЕ ПРОЕКТА "Project SixSeven"
Данный проект представляет собой веб сервер, который отправляет команды контроллеру Teensy, который умеет поднимать и опускать платформу. Веб сервер создан с упростить общение с котроллером и дать возможность любому человеку управлять положением платформы. Если при запуске выдается ошибка ненайденного устройства по данному порту, то нужно изменить зачение переменной SERIAL_PORT на название порта, к которому подключен микроконтроллер.
## Запуск
1. Убедитесь что у вас установлен pyserial и python (команды для установки: pip instal python3; pip install pyserial)
2. В командной строке откройте директорию с репозиторием
3. Введите python3 server.py в командную строку
4. Откройте в бразуре ссылку localhost:8000
## Управление
1. При нажатии кнопки "Калибровать" текущее положение объявляется нулевым.
2. Кнопка "В ноль" возвращает платформу в нулевое положение.
3. Управлять положением можно при помощи ползунка. Каждый ползунок вращает свой двигатель, в результате чего платформа подниматеся или опускается. Определить текущее положение платформы можно по показателю под ползунками.
