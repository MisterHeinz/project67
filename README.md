# ОПИСАНИЕ ПРОЕКТА "Project SixSeven"
Данный проект представляет собой веб сервер, который отправляет команды контроллеру Teensy, который умеет поднимать и опускать платформу. Веб сервер создан с упростить общение с котроллером и дать возможность любому человеку управлять положением платформы. Данный web-сервер и общение с контроллером работают только на unix-подобных системах. Для поддержки Windows нужно в поменять значение в константе SERIAL_PORT на COM3.
## Запуск
1. Убедитесь что у вас установлен pyserial и python (команды для установки: pip instal python3; pip install pyserial)
2. В командной строке откройте директорию с репозиторием
3. Введите python3 server.py в командную строку
4. Откройте в бразуре ссылку localhost:8000
## Управление
1. При нажатии кнопки "Калибровать" текущее положение объявляется нулевым.
2. Кнопка "В ноль" возвращает платформу в нулевое положение.
3. Управлять платформой можно при помощи расположенных по бокам кнопок "+" и "-". Каждая стороная является самостоятельной осью. 
4. Под кнопками "+" и "-" есть показатель текущего положения. Нажав на него можно ввести нужное вам положение.
